# oucru_tutorials

This is a repository with a series of tutorials developed for researchers at [OUCRU]. If you aren't from OUCRU, they might still be useful. If you are from OUCRU, and you come across any problems using the materials, raise an issue in the github repo. Thanks!

[OUCRU]: http://www.tropicalmedicine.ox.ac.uk/vietnam-oucru
